# co-logistic-service

## Description

Microservice implemented in NestJS as microservice, using the request-response message style to exchange messages with other microservices. 
Simulates the creation of a logistic sent order.
Uses port 3003.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
